# Sito web di ASAI-ER

## Come contribuire

### Testare il sito in locale

1. Installare [hugo](https://gohugo.io/), versione extended
2. Clonare questo repository
3. aprire un terminale nella cartella clonata
4. lanciare `hugo server`. Il sito risponderà su localhost:1313 e mostrerà le modifiche in tempo reale

### Aggiungere o modificare contenuti

* Contatti: modificare `data/contact.yaml`
* Specchietti con le discipline: modificare `data/features.json`
* Venue, docenti, programma, pagina di iscrizioni: sono i relativi file Markdown in `content/services`
* A modifiche fatte, aggiungerle al tracker di git (`git add...`), quindi fare un normale commit e una push
    * **per piacere**: usate commit messages sensati.
